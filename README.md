# CSCI3170 project

### how to run
#### TRY THIS FIRST
Windows:
```
Ctrl + Shift + B
```

MacOS:
```
⌘ + ⇧ + b
```

Windows:
```
javac -d ./out/ -classpath ".;ojdbc7.jar" "./csci3170project/*.java"
java -classpath ".;ojdbc7.jar" csci3170project.Project
```

Linux/Mac:
```
javac -d ./out/ -classpath ".:ojdbc7.jar" "./csci3170project/*.java"
java -classpath ".:ojdbc7.jar" csci3170project.Project
```


Old stuff
```
javac Project.java 
java -classpath ./ojdbc7.jar:./ Project
```
